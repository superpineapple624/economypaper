## 📊 文献表格 - 科创板制度改革的效果——基于股票定价效率、流动性和上市公司质量的研究

| 维度      | 内容                                                                                    |
| ------- | ------------------------------------------------------------------------------------- |
| 论文标题    | 科创板制度改革的效果——基于股票定价效率、流动性和上市公司质量的研究                                                    |
| 作者      | 张晓燕、张子健                                                                               |
| 发表年份    | 2022年                                                                                 |
| 研究问题    | 探究科创板发行承销、市场化交易制度的改革措施，对股票定价效率、交易流动性以及上市公司质量的影响。                                      |
| 研究方法    | 以科创板公司为实验组、主板公司为对照组，按照行业、IPO前一年总资产、公司年龄进行样本匹配，控制相关财务变量降低选择性偏差，通过回归分析检验假设，同时开展多轮稳健性检验。 |
| 主要发现    | 相比对照组，科创板股票在IPO和正常交易阶段定价效率更高，交易价格冲击更小，交易价差平均低2.9个基本点；科创板公司上市后研发投入、盈利等增长有所提升，但效果并不显著。  |
| 创新点     | 是较早系统研究科创板制度改革成效的成果，覆盖两类改革对三个维度的影响；采用多重方法降低样本选择性偏差，包括样本匹配、多维度控制、多场景稳健性检验，结论可靠性更强。     |
| 局限性     | 仅覆盖科创板公司上市后1年内的财务数据，无法判断公司质量的长期表现；进一步控制选择性偏差后部分结果显著性下降，受样本量限制较大；未完全拆解不同改革措施的单独效应。     |
| 与本研究的关联 | 未提及                                                                                   |
