## AI 总结 - 科创板制度改革的效果——基于股票定价效率、流动性和上市公司质量的研究

### 核心内容总结

## 这篇论文聚焦科创板注册制试点的改革效果，以2019年7月至2020年10月的188家科创板上市公司为实验组、同期上交所主板上市公司为对照组，通过“行业+IPO前总资产+公司年龄”的一对一匹配，搭配控制影响上市选择的财务指标，缓解公司自主选择上市板块带来的自选择偏差，系统评估了科创板在发行承销、市场化交易两类制度改革对股票**IPO阶段、正常交易阶段的定价效率**、**交易流动性**，以及**上市后公司财务质量**的影响。研究发现：科创板改革显著提升了两个交易阶段的定价效率，降低了交易价格冲击，平均交易价差比主板低2.9个基点；但科创板公司上市后研发投入、盈利的增长提升效果暂不显著。

## 一、研究背景与制度背景

### 1. 研究背景

科创板2018年11月提出设立、2019年7月正式开板，是我国资本市场注册制改革的首个试验田，核心目标一是支持科创企业融资，二是在发行承销、信披、交易、并购重组、退市五个领域试点市场化改革。此前国内研究已经证实主板的IPO限价、10%涨跌停等制度会降低定价效率、加剧价格扭曲，但尚未有研究系统评估科创板改革的实际效果，本文填补了这一空白。

### 2. 科创板与主板的制度差异

论文重点对比了两类核心改革的差异：

#### （1）发行承销制度

| 维度   | 科创板                                     | 主板                                |
| ---- | --------------------------------------- | --------------------------------- |
| 行业要求 | 必须符合科创定位的高新技术行业，额外要求科创属性                | 无特殊要求                             |
| 发行条件 | 允许未盈利、存在累计未弥补亏损的企业上市，参考预计市值、研发投入等多元指标   | 对净利润、营收、现金流有严格硬性要求，不允许未盈利/未补亏企业上市 |
| 发行定价 | 完全市场化，无价格上限                             | 存在23倍市盈率的隐形上限                     |
| 审核制度 | 注册制：上交所审核上市条件和信披合规性，证监会仅做注册程序，交易所不做价值判断 | 核准制：证监会对发行条件、投资价值做实质性审核和价值判断      |
| 信息披露 | 要求内容通俗易懂，明确披露募资用途，额外要求披露审计报告、法律意见等文件    | 仅要求完整真实，对募资使用、锁定期安排无强制披露要求        |


#### （2）交易制度

| 维度                                           | 科创板                                      | 主板                               |
| -------------------------------------------- | ---------------------------------------- | -------------------------------- |
| 投资者门槛                                        | 前20交易日日均资产≥50万+24个月交易经验                  | 无门槛                              |
| 涨跌幅规则                                        | 上市前5个交易日无涨跌幅限制，正常交易涨跌幅20%                | IPO首日涨幅上限144%、跌幅下限64%，正常交易涨跌幅10% |
| 申报规则                                         | 设置“价格笼子”：买入申报价不高于基准价102%，卖出不低于98%，防范价格操纵 | 无价格范围限制                          |
| 融资融券                                         | 上市首日即可纳入两融标的                             | 上市满3个月后才可能纳入两融标的                 |
| 论文认为上述改革理论上会提升股票定价效率、改善流动性，因此从这三个维度出发评估改革效果。 |                                          |                                  |

## 二、研究设计

### 1. 识别策略：解决自选择偏差

企业选择上科创板还是主板并非随机，科创企业本身的特征就和主板企业存在差异，直接回归会导致估计偏差。论文使用潜在结果框架下的平均处理效应（ATT）进行识别：

$\displaystyle ATT = E(Y_{i1} \mid STAR_i = 1) - E(Y_{i0} \mid STAR_i = 1)$

\
其中

$STAR_i=1$

表示公司

$i$

在科创板上市，

$Y_{i1}$

是科创板上市的结果变量表现，

$Y_{i0}$

是该公司如果在主板上市的潜在表现，但后者无法观测。如果直接用科创板和主板的观测值做差，会包含选择偏差：\


$\displaystyle E(Y_{i1} \mid STAR_i=1) - E(Y_{i0} \mid STAR_i=0) = ATT + SelectionBias$

\
为了消除选择偏差，论文假设在控制影响上市选择的变量

$X_i$

后，上市板块选择和潜在结果满足条件独立，因此使用如下回归模型估计：\


$\displaystyle Y_i = b_0 + b_1 STAR_i + b_2' X_i + \varepsilon_i$

\
其中

$b_1$

就是科创板改革的净效应。

### 2. 样本匹配

为了满足条件独立假设，论文首先对样本进行匹配：

*   实验组：2019.7.22-2020.10.26上市的188家科创板公司

*   对照组：2014.1.1-2020.10.26上交所主板上市的602家非金融公司\
    匹配规则为：同一行业内，按IPO前一年总资产、公司年龄的标准化距离匹配，距离公式为：\
    $\displaystyle dist_{j,i}^k = \frac{|x_{j,k}^{TA} - x_{i,k}^{TA}|}{\sigma_k^{TA}} + \frac{|x_{j,k}^{Age} - x_{i,k}^{Age}|}{\sigma_k^{Age}}$\
    其中 $\sigma_k^{TA}$ 、 $\sigma_k^{Age}$ 是行业 $k$ 内主板公司总资产、年龄的标准差，选择距离最小的主板公司和科创板公司配对，允许一个主板公司匹配多个科创板公司。\
    匹配后共得到187对有效样本，匹配前科创板平均总资产25.23亿、主板44.63亿，差异在1%水平显著；匹配后主板平均总资产20.41亿，和科创板的25.32亿无显著差异，公司年龄也无显著差异，说明匹配有效。匹配后回归额外控制IPO前一年的扣非净利润、营业收入、经营现金流、未分配利润四个影响上市决策的财务指标，进一步降低偏差。

## 三、研究假设与变量设计

论文提出四个研究假设，对应三类被解释变量：

### 1. 假设一：科创板改革提升了IPO阶段的定价效率

IPO阶段定义为股票上市后20个交易日，使用三个指标衡量定价效率：

*   $Corr^{IPO}$ ：IPO阶段个股日收益率的一阶自相关系数绝对值，数值越小说明信息越快融入当日价格，定价效率越高；

*   $Stopnum$ ：上市后连续涨停天数，越少说明定价效率越高。由于科创板前5日无涨跌幅限制，无法直接观测该指标，因此通过主板涨跌停规则反推： $(1+44\%)(1+10\%)^{Stopnum_{i,STAR}-1} = R_{i,STAR}^{IPO}$ ，其中 $R_{i,STAR}^{IPO}$ 是科创板IPO首日收益率；

*   $Volatility^{IPO}$ ：IPO阶段日收益率的波动率，越高说明交易越活跃，信息融入价格越充分，定价效率越高。

### 2. 假设二：科创板改革提升了正常交易阶段的定价效率

正常交易阶段定义为上市20个交易日之后，使用两类指标衡量定价效率：

#### （1）滞后效率指标（Hou and Moskowitz, 2005）

首先将个股日收益率对同期市场收益率、滞后4期市场收益率回归：

$\displaystyle r_{i,t} = \alpha_i + \beta R_{m,t} + \sum_{n=1}^4 \delta^n R_{m,t-n} + \varepsilon_{i,t}$

*   $Delay1 = 1 - \frac{R^2_{\delta^n=0, \forall \delta \in [1,4]}}{R^2}$ ：即限制滞后项系数为0的回归拟合优度除以全模型拟合优度，数值越小说明滞后市场信息对个股收益的解释力越低，定价效率越高；

*   $Delay2 = \frac{\sum_{n=1}^4 |\delta^n|}{|\beta| + \sum_{n=1}^4 |\delta^n|}$ ：滞后项系数绝对值占总系数绝对值的比例，越小说明定价效率越高。

#### （2）方差比例指标（Lo and Mackinlay, 1988）

$\displaystyle VR(k,m) = \left| \frac{Var(r_{i,t+1:t+m})/m}{Var(r_{i,t+1:t+k})/k} - 1 \right|$

\
若收益率无序列相关，不同区间的方差比值应接近1，因此VR越小说明定价效率越高，论文计算了短期

$VR(1,2)$

和长期

$VR(1,5)$

两个指标。

### 3. 假设三：科创板改革提升了正常交易阶段的流动性

使用三个流动性指标：

*   日换手率 $Turnover$ ：日交易量除以A股流通股本，越高流动性越好；

*   Amihud指标： $Amihud_{i,t} = \sum_{n=1}^N \frac{|r_{i,t+n}|}{Amount_{i,t+n}} / N$ ，衡量单位成交额带来的收益率变动，越小说明价格冲击越小，流动性越好；

*   Roll价差： $Rollsprd_{i,t} = 2 \times \sqrt{-cov(r_{i,t}, r_{i,t+1})}$ ，衡量隐性买卖价差，越小说明交易成本越低，流动性越好。

### 4. 假设四：科创板上市后企业的研发、盈利增长更快

## 使用季度环比增长率作为被解释变量：研发支出增速$g\_R\&D$、扣非净利润增速$g\_Profit$、营业收入增速$g\_Revenue$、经营现金流增速$g\_Cash$，同时用同比增速做稳健性检验。

## 四、实证结果

### 1. 基准回归结果

#### （1）IPO阶段定价效率检验

$STAR_i$对$Corr^{IPO}$的系数为-0.073，在1%水平显著；对$Stopnum$的系数为-4.68，在1%水平显著，说明科创板IPO阶段的定价效率显著高于主板，连续涨停天数平均少4.68天；对$Volatility^{IPO}$的系数显著为正，说明科创板IPO阶段交易更活跃，信息融入更充分，假设一成立。

#### （2）正常交易阶段定价效率检验

$STAR_i$对$Delay1$的系数为-0.007，对$Delay2$的系数为-0.006，均在1%水平显著，说明科创板股票对当期市场信息反应更快，滞后信息的影响更小；但对两个VR指标的系数不显著，整体说明正常交易阶段定价效率有一定提升，假设二部分成立。

#### （3）流动性检验

$STAR_i$对$Turnover$的系数为-0.006，在1%水平显著，说明科创板日换手率比主板低0.6个百分点；对Amihud的系数显著为负（10%水平），说明价格冲击更小；对$Rollsprd$的系数为-0.029，在1%水平显著，说明科创板平均交易价差比主板低2.9个基点，整体流动性更好，假设三成立。

#### （4）上市公司质量检验

$STAR_i$对研发增速、利润增速、营收增速的系数均为正，但统计上不显著；对现金流增速的系数为负也不显著，说明科创板公司上市后财务质量的提升效果暂不明显，可能是因为样本仅包含上市后1年的财务数据，长期效应尚未显现，假设四不成立。

### 2. 拓展研究：融资融券的影响

科创板上市首日即可两融，而主板上市满3个月才可能纳入两融标的，论文进一步检验两融交易对科创板效率和流动性的影响：仅使用科创板样本，以滞后1个月的融资交易额占比$Margin$、融券交易额占比$Short$作为自变量回归，结果发现：

*   融资交易占比越高， $Delay1$ 越低（定价效率越高），同时换手率越高、Amihud越低（流动性越好）；

*   融券交易占比越高， $VR(1,5)$ 越低（定价效率越高），但对流动性无显著影响，原因是科创板融券交易额占比仅为2%，远低于融资的7.31%，卖空限制导致融券的作用未充分发挥。

### 3. 稳健性检验

#### （1）进一步缓解自选择偏差

部分科创板企业同时满足主板上市条件，其上市选择可能存在未观测的偏差，因此论文筛选出仅满足科创板上市条件、不满足主板条件的32家科创板企业，和科创板设立前（2014-2019年7月）的主板公司匹配后重新回归，结果显示科创板IPO定价效率、流动性仍然更高，只是正常交易阶段定价效率的显著性下降，主要原因是样本量过小。

#### （2）更换对照组为创业板

## 创业板同样服务科创类企业，2020年8月才落地注册制改革，因此论文使用2014-2020年8月的创业板公司作为对照组重新匹配回归，结果和基准回归完全一致，说明结论稳健。

## 五、结论

科创板的发行承销、市场化交易制度改革，确实显著提升了股票在IPO和正常交易阶段的定价效率，降低了交易价格冲击和交易成本，证明了市场化改革的有效性；但上市公司在研发投入、盈利增长方面的提升效果暂不显著，需要更长时间的样本观测。研究结论为后续注册制向全市场推广、完善资本市场市场化交易制度提供了实证支持。
