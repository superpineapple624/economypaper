## 📊 文献表格 - 科创板改革对新股抑价和股票流动性的影响分析

| 维度      | 内容                                                                                                                                                                |
| ------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 论文标题    | 科创板改革对新股抑价和股票流动性的影响分析                                                                                                                                             |
| 作者      | 邹文博                                                                                                                                                               |
| 发表年份    | 2023年                                                                                                                                                             |
| 研究问题    | 以2019年科创板制度改革为场景，检验科创板取消市盈率限制、提升个人投资者准入门槛两项改革对新股抑价程度、股票流动性的影响，同时探究信息风险在上述影响中的调节效应。                                                                                |
| 研究方法    | 采用实证研究方法，将科创板上市股票作为实验组，存在发行市盈率限制的主板、创业板上市股票作为对照组，构建OLS回归模型检验改革效应，加入交互项验证调节作用，同时选取2018年后上市样本开展稳健性检验，数据来自CSMAR数据库。                                                  |
| 主要发现    | 1. 科创板市场化定价显著降低了一级市场新股抑价程度，其上市股票长期累计收益率较主板低约80%；2. 科创板提升个人投资者准入门槛降低了板块股票流动性，长期换手率较主板低20%-25%，缓解了A股换手率过高的问题；3. 信息风险存在调节效应，信息不对称程度更低的大公司、高资产收益率公司在科创板上市后的收益率和流动性更高。 |
| 创新点     | 1. 利用科创板改革这一外生事件，验证了市场化定价改革对缓解新股抑价的作用，拓展了发行制度市场化改革的相关研究；2. 从个人投资者准入门槛的角度，补充了个人投资者参与对股票流动性影响的研究视角；3. 厘清了信息风险在科创板改革影响新股定价和流动性过程中的调节效应。                              |
| 局限性     | 限于研究侧重点与篇幅，未进一步探讨科创板改革降低流动性对公司治理、股价崩盘风险等维度的影响，也未深入评估流动性下降对市场长期效率的作用。                                                                                              |
| 与本研究的关联 | 未提及                                                                                                                                                               |
