## AI 总结 - 市场化改革提升了定价效率吗？——来自注册制的证据

### 论文核心内容总结

## 这篇《市场化改革提升了定价效率吗？——来自注册制的证据》以2018年11月5日科创板试点注册制宣告这一A股资本市场市场化改革标志性准自然实验为切入点，从科创板新股定价表现、壳企业市场反应、非科创板新股定价变化三个维度展开实证检验，最终证明注册制通过引导投资者形成理性预期，有效抑制了A股长期存在的“炒新”“炒壳”“炒概念”等异象，显著提升了市场定价效率，为我国后续资本市场市场化改革提供了实证支撑。

### 一、研究背景与问题提出

A股长期实施核准制，监管部门对拟上市企业做实质性价值判断，同时存在上市审批周期长、供给管制、退市制度不完善等问题，催生了三大市场乱象：

1.  **IPO抑价与炒新**：新股发行稀缺，叠加2014年出台的首日44%涨幅限制，新股上市后往往连续多日涨停，实际首日收益率畸高，“打新”成为无风险收益。
2.  **炒壳成风**：上市资格稀缺，绩差壳公司因拥有“上市牌照”获得估值溢价，价格与基本面完全脱节，借壳上市、保壳重组等行为扭曲资源配置。
3.  **炒概念盛行**：小盘股、科技股、媒体追捧的个股因容易控盘，成为资金炒作对象，估值严重偏离基本面。\
    2018年科创板试点注册制拉开了A股市场化改革的序幕，注册制将监管重心从“事前资质审批”转向“事中事后信息披露监管”，取消盈利要求、放开定价管制、配套严格退市制度，为研究市场化改革对定价效率的影响提供了绝佳的研究场景，论文围绕“注册制是否提升了A股定价效率”这一核心问题展开研究。

***

### 二、理论分析与研究假设

论文从三个维度提出研究假设：

#### 假设1：科创板新股定价效率更高

注册制在科创板首先试点，三大制度设计提升定价效率：

1.  取消发行盈利限制、扩大上市供给，降低了新股的稀缺性溢价；
2.  引入专业机构询价、取消市盈率管制，发行定价权交还市场，价格更能反映企业基本面；
3.  上市前5日不设涨跌幅、后续涨放宽至20%，价格发现机制更顺畅，新股价格更快回归合理价值。\
    因此**相比非科创板，科创板IPO实际首日收益率更低，连续涨停天数更少**。

#### 假设2：注册制宣告后壳企业市场表现更差

注册制落地后：

1.  上市门槛降低、周期缩短，企业借壳上市的动机大幅下降，壳资源的稀缺性价值消失；
2.  退市制度趋严，壳企业退市风险大幅提升，投资者预期壳公司不再具备“牌照价值”。\
    同时注册制全面推广是明确趋势，炒壳的长期逻辑被打破，因此**注册制宣告后，壳企业的短期和长期市场表现都更差**。

#### 假设3：注册制宣告后非科创板新股定价效率也会提升

## 尽管注册制初期仅在科创板实施，但投资者会形成“未来注册制全面推广、A股市场扩容、新股供给增加、炒新收益下降”的理性预期，会提前调整投资行为，减少对核准制下新股的炒作，因此**注册制宣告后，核准制下的非科创板IPO实际首日收益率更低，连续涨停天数更少**。

### 三、研究设计

#### 1. 样本选择

论文分三个研究场景匹配样本：

*   科创板与非科创板新股对比：选取2019年7月22日（科创板首批上市）至2020年7月22日的IPO样本，剔除金融行业、数据缺失样本，上下1%缩尾后得到262个观测值。
*   注册制宣告的壳企业事件研究：选取2018年11月5日（注册制宣告日）前后的A股上市公司样本，剔除窗口期停牌超过1个月、金融行业样本，缩尾后得到2109个观测值。
*   注册制宣告前后非科创板新股对比：选取2014年1月1日（2013年底首日限价政策实施）至2020年2月10日的非科创板IPO样本，剔除金融行业、豁免首日限价的样本，缩尾后得到1193个观测值。

#### 2. 核心变量定义

| 变量类型        | 变量符号               | 变量定义                                                                                                                                 |
| ----------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------ |
| 被解释变量（定价效率） | $IR$（实际首日收益率）      | 新股首个收盘未涨停日的收盘价与发行价的涨跌幅，规避首日44%涨幅限制的干扰，反映真实炒作幅度                                                                                       |
|             | $LU$（连续涨停天数）       | 新股上市首日至首个未涨停日的连续涨停天数；科创板前5日无涨跌幅，通过公式反推限价规则下的涨停天数：$LU=\mathrm{Ln}\left( \frac{1+IR}{1.44} \right)/\mathrm{Ln}1.1 +1$，$IR<44\%$时$LU=0$ |
| 被解释变量（市场反应） | $CAR$（累计超额收益率）     | 事件窗口期内个股实际收益减去预期收益的累计值，衡量短期市场反应                                                                                                      |
|             | $BHAR$（购买并持有超额收益率） | 事件后60/120日个股买入持有收益减去市场收益，衡量长期市场反应                                                                                                    |
| 核心解释变量      | $STAR$             | 科创板虚拟变量，企业在科创板上市取1，否则取0                                                                                                              |
|             | $Announ$           | 注册制宣告虚拟变量，非科创板企业在2018年11月5日后上市取1，否则取0                                                                                                |
|             | $ESV/MV$           | 壳价值含量，壳价值除以公司总市值，值越高说明壳价值占比越高                                                                                                        |
|             | $Shell90/Shell95$  | 壳企业虚拟变量，$ESV/MV$超过90%/95%分位数取1，否则取0                                                                                                  |
| 控制变量        | -                  | 上市前一年资产负债率、净资产收益率、发行规模、市场情绪、行业固定效应、IPO年份固定效应等                                                                                        |


#### 3. 实证模型

1.  检验科创板新股定价效率：\
    $\displaystyle PricingEfficiency_i = \alpha + \beta STAR_i + \lambda Controls_i + IndFE_i + YearFE_i + \varepsilon_i$\
    若 $\beta$ 显著为负，说明科创板定价效率更高，验证假设1。

2.  检验注册制对壳企业的影响：\
    $\displaystyle MarketResponse_i = \alpha + \beta X_i + \lambda Controls_i + IndFE_i + \varepsilon_i$\
    其中 $X_i$ 为 $ESV/MV$ 或 $Shell90/Shell95$ ，若 $\beta$ 显著为负，说明壳企业在注册制宣告后表现更差，验证假设2。

3.  检验注册制对非科创板新股的影响：\
    将第一个模型的 $STAR$ 替换为 $Announ$ ，若 $\beta$ 显著为负，说明注册制宣告后非科创板新股定价效率提升，验证假设3。

***

### 四、实证结果

#### 1. 基础假设验证

*   **假设1验证**：回归结果显示 $STAR$ 的系数在1%水平显著为负，经济意义上科创板新股的实际首日收益率比非科创板低39.51%，连续涨停天数少28.36%，科创板定价效率显著更高。

*   **假设2验证**：短期来看，注册制宣告后壳价值越高的公司累计超额收益率 $CAR$ 越低，壳企业的短期表现比非壳企业差；长期来看，事件后60日、120日的购买并持有超额收益率 $BHAR$ 同样随壳价值升高而显著降低，说明注册制对壳企业的负面冲击具有持续性，假设2成立。

*   **假设3验证**： $Announ$ 的系数在1%水平显著为负，注册制宣告后，非科创板新股的实际首日收益率下降38.97%，连续涨停天数下降35.37%，说明注册制的预期效应已经传导到核准制板块，非科创板新股的定价效率也显著提升。\
    论文还做了一系列稳健性检验：更换事件窗口期、用Fama-French三因子/四因子/五因子模型重新计算超额收益、用倾向得分匹配（PSM）匹配宣告前后特征相似的IPO样本、做安慰剂检验（假设注册制宣告在2015/2016/2017/2018年初，结果核心变量系数不显著或为正），均验证了结论的可靠性。

#### 2. 异质性检验（进一步检验）

论文进一步验证注册制对炒作程度更高的个股抑制作用更明显：

1.  **对概念股的抑制**：发行规模越小、中签率越低的个股“炒新”效应越强，交乘项 $Announ \times 小盘股$ 、 $Announ \times 低中签率$ 的系数均显著为负，说明注册制对概念股炒作的抑制作用更强。

2.  **对科技股的抑制**：科技股此前因稀缺性被资金追捧，交乘项 $Announ \times 高科技行业$ 系数显著为负，注册制放开科技企业上市门槛后，科技股稀缺性下降，炒作被抑制。

3.  **对媒体追捧股的抑制**：上市前媒体正面报道越多的个股越容易被炒作，交乘项 $Announ \times 媒体正面度$ 系数显著为负，说明注册制下投资者不再盲目追涨媒体热炒的个股。

4.  **对次新股的抑制**：注册制宣告前1年、1.5年、2年上市的次新股，在宣告后估值也显著下降，炒次新的情绪被缓解。

#### 3. 补充事件检验

## 论文还检验了2019年3月科创板规则正式发布、2019年12月新证券法修订（明确全面推行注册制）、2020年创业板注册制改革等相关事件，结果均显示壳企业市场表现显著变差、新股定价效率提升，和核心结论一致。

### 五、结论与启示

#### 核心结论

注册制从三个维度提升了A股定价效率：①科创板新股定价效率显著高于核准制板块；②注册制通过改变投资者预期，让壳资源价值大幅下降，炒壳行为被抑制；③注册制的政策外溢效应提升了核准制板块的新股定价效率，同时对概念股、科技股、媒体追捧股的炒作抑制作用更明显。整体来看注册制减少了行政干预，引导投资者预期回归基本面，有效提升了市场定价效率。

#### 实践启示

1.  注册制改革方向正确，要坚持让市场在资源配置中发挥决定性作用，监管部门侧重信息披露监管，减少对发行定价、交易的行政干预，避免注册制流于形式。
2.  A股投资者结构、市场文化和境外成熟市场差异较大，不能完全照搬境外制度，改革过程中要充分考虑中国市场实际，稳步推进。
3.  注册制改革的预期引导作用显著，后续政策要保持透明度和连续性，进一步强化投资者的理性预期，持续抑制市场炒作乱象。
