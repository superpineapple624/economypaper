## 📊 文献表格 - 股票转让方式与极端收益风险——兼论新三板市场引入做市转让制度的经济后果

| 维度      | 内容                                                                                                                                            |
| ------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| 论文标题    | 股票转让方式与极端收益风险：新三板引入做市商制度的经济后果                                                                                                                 |
| 作者      | 陈辉、顾乃康                                                                                                                                        |
| 发表年份    | 2017年                                                                                                                                         |
| 研究问题    | 现有极端收益风险生成机制研究普遍忽略交易环节的作用，做市转让制度实施效果的相关实证研究也未关注其对极端收益风险的影响，本文探究新三板不同股票转让方式与极端收益风险的关系及作用路径。                                                    |
| 研究方法    | 以新三板挂牌公司为研究样本，采用Logit模型、Poisson模型进行基准回归，使用处理效应模型控制自选择偏差，同时开展异质性分析、机制检验和多维度稳健性检验。                                                              |
| 主要发现    | 相较协议转让股票，做市转让股票的极端收益风险显著更低，但做市商数量增加不会进一步显著降低极端收益风险。转让方式对股价暴涨风险的抑制作用在熊市更强，对股价暴跌风险的抑制作用在牛市更强。转让方式对极端收益风险的影响主要通过股票流动性路径实现，信息有效性路径因与流动性高度相关才发挥作用。 |
| 创新点     | 从交易环节视角提出解释极端收益风险生成机制的新假说，首次实证检验了新三板做市转让制度对极端收益风险的影响，厘清了其作用路径，丰富了相关领域研究文献。                                                                    |
| 局限性     | 未提及                                                                                                                                           |
| 与本研究的关联 | 该研究深化了学界对极端收益风险生成机制的认知，为完善新三板交易机制提供了实证支撑，也为后续股票转让制度与极端收益风险相关研究提供了参考。                                                                          |
