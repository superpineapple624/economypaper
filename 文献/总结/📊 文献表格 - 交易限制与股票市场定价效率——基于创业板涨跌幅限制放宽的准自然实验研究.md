## 📊 文献表格 - 交易限制与股票市场定价效率——基于创业板涨跌幅限制放宽的准自然实验研究

| 维度      | 内容                                                                                              |
| ------- | ----------------------------------------------------------------------------------------------- |
| 论文标题    | 交易限制与股票市场定价效率：基于创业板注册制改革的准自然实验                                                                  |
| 作者      | 顾明、曾力、陈海强、倪博，均任职于厦门大学经济学院/王亚南经济研究院                                                              |
| 发表年份    | 2022年                                                                                           |
| 研究问题    | 探究2020年8月24日创业板涨跌幅限制从10%放宽至20%的改革能否在不加剧市场不稳定的前提下提升市场定价效率，厘清涨跌幅限制放松的作用机制。                        |
| 研究方法    | 采用准自然实验设计，以创业板股票为处理组、PSM匹配的中小板股票为对照组构建双重差分模型检验政策效果；辅以事件研究、异质性分析，使用5分钟高频数据验证涨跌停附近的微观交易机制。        |
| 主要发现    | 涨跌幅限制放松显著提升了市场定价效率，股价对公共信息更敏感且融入了更多公司异质性信息；信息透明度更低的企业定价效率改善更明显；政策缓解了交易干扰，降低了磁吸效应、日内波动率聚类和波动率溢出。 |
| 创新点     | 以创业板涨跌幅限制调整这一纯外生政策冲击为研究场景，有效缓解了过往研究的内生性问题；从市场、公司、微观交易三个层面多维度验证政策效果，明确了涨跌幅限制影响定价效率的具体路径。         |
| 局限性     | 未提及                                                                                             |
| 与本研究的关联 | 未提及                                                                                             |
