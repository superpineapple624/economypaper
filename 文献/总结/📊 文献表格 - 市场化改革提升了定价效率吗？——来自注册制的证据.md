## 📊 文献表格 - 市场化改革提升了定价效率吗？——来自注册制的证据

| 维度      | 内容                                                                                                      |
| ------- | ------------------------------------------------------------------------------------------------------- |
| 论文标题    | 市场化改革提升了定价效率吗？——来自注册制的证据                                                                                |
| 作者      | 赖黎、蓝春丹、秦明春                                                                                              |
| 发表年份    | 2022年                                                                                                   |
| 研究问题    | 聚焦科创板试点注册制这一A股市场化改革事件，探究注册制的推行是否提升了我国资本市场定价效率，厘清注册制影响定价效率的作用逻辑。                                         |
| 研究方法    | 采用事件研究法，通过累计超额收益率、购买并持有超额收益率衡量市场反应；使用单变量检验、多元回归分析对比不同板块、注册制前后的新股定价效率；辅以倾向得分匹配、安慰剂检验、异质性分析等方法保障结论稳健。     |
| 主要发现    | 科创板新股实际首日收益率、连续涨停天数显著低于非科创板；注册制宣告后壳企业长短期市场表现更差，非科创板新股的首日收益率、连续涨停天数也显著下降；注册制对概念股、科技股、媒体追捧股的炒作抑制作用更明显。    |
| 创新点     | 以注册制宣告外生事件为研究契机，补充了市场化改革影响A股定价效率的相关文献；从投资者预期视角揭示了注册制抑制投机、提升定价效率的内在机制；验证了注册制下A股股价信息含量提升，为后续市场化改革提供了经验参考。 |
| 局限性     | 仅考察了注册制推行初期的政策效果，注册制在全市场全面落地后的长期影响有待后续进一步研究验证。                                                          |
| 与本研究的关联 | 该文系统验证了注册制对资本市场定价效率的提升作用，为注册制改革效果评估、资本市场市场化改革相关研究提供了实证依据和研究思路。                                          |
