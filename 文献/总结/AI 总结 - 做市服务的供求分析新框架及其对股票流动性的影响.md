## AI 总结 - 做市服务的供求分析新框架及其对股票流动性的影响

### 核心内容总结

## 这是2017年发表于《财贸经济》的论文，作者陈辉，聚焦新三板（全国中小企业股份转让系统）的做市转让制度，首次从供需匹配的视角构建了做市商服务的分析框架，既探究了挂牌企业转让方式选择、做市商数量的决定因素，也验证了做市转让制度对股票流动性的实际影响。研究发现：企业的转让方式选择是供需双方共同作用的结果，流动性越好、市值越高、股本越小、股价越高的新三板企业更倾向于选择做市转让；而企业规模越大、股价波动率越低，愿意为其提供服务的做市商数量越多。同时，挂牌企业从协议转让转为做市转让后，股票流动性会显著提升，该结果在控制自选择偏差等内生性问题后依然稳健，最终论文也针对新三板制度优化提出了相应政策建议。

### 一、研究背景与问题提出

## 新三板在2014年8月正式推出做市转让制度，此前挂牌企业仅能采用协议转让方式，市场整体流动性极差，监管层期望通过做市商制度改善流动性。但制度运行后出现两个典型现象：一是仅不足20%的挂牌企业选择做市转让，多数企业仍然选择协议转让；二是不同做市企业的做市商数量差异极大，部分企业仅有2家做市商，部分企业的做市商数量超过数十家。 既有研究要么仅从企业需求侧分析转让方式选择，要么仅从做市商供给侧分析做市决策，没有将二者结合起来理解做市服务的匹配逻辑，同时针对新三板做市制度效果的研究也大多没有解决“流动性好的企业更愿意选做市”的自选择偏差问题，结论可靠性不足，这是本文的研究切入点。

### 二、理论框架与研究假说

本文参考经典做市商成本模型，构建了做市服务的供需匹配框架：

#### 1. 做市商的成本函数

参考Stoll(1978)的做市商定价模型，做市商为股票$i$提供做市服务的总成本为：

$\displaystyle s_i = z\tau_i\sigma_i^2 \left|Q_i\right|/W_0 + 2a_i + 2M/\left|Q_i\right|$

\
其中：

*   $z$ 为做市商的风险厌恶系数， $\tau_i$ 为做市商持有该股票的平均时长， $\sigma_i^2$ 为股票收益率的波动率， $\left|Q_i\right|$ 为做市商的最优存货头寸， $W_0$ 为做市商初始财富，第一项代表做市商的存货风险成本；

*   $a_i$ 为单只股票的固定做市成本（包括系统接入、研究投入等）， $M$ 为单笔交易的可变成本，后两项分别代表固定成本和可变交易成本。\
    企业愿意为做市服务支付的最高对价为 $s^*$ ，即做市带来的流动性提升、估值溢价等总收益，供需匹配存在三种均衡状态：\
    $\displaystyle \begin{cases} n=1, P_i \geq 0, & \text{if } s_i < s^* \quad (\text{做市成本低于企业支付意愿，企业选择做市})\\ n=1, P_i \geq z\tau_i\sigma_i^2 |Q_i|/W_0 + 2a_i + 2M/|Q_i| - s_i, & \text{if } s_i = s^* \quad (\text{供需刚好匹配，企业选择做市})\\ n=0, P_i < z\tau_i\sigma_i^2 |Q_i|/W_0 + 2a_i + 2M/|Q_i| - s_i, & \text{if } s_i = s^* \quad (\text{做市成本高于支付意愿，企业选协议转让}) \end{cases}$\
    &#x20;$n$ 为企业是否选择做市的虚拟变量， $P_i$ 为做市服务的净收益。

#### 2. 研究假说

基于上述模型，本文提出两类假说：

##### （1）转让方式选择的影响因素（供需共同作用）

*   流动性：企业原有流动性越高，一方面做市的边际收益越低（需求侧负向影响），另一方面做市商的做市成本越低（供给侧正向影响），最终影响由二者合力决定；
*   企业规模：市值越高的企业，做市带来的流动性溢价总价值越高（需求侧正向），同时做市商的价差收入空间更大（供给侧正向），因此更有可能选择做市；
*   波动率：股价波动率越高，做市商的存货风险成本越高（供给侧负向），企业需要支付的做市成本越高，越难选择做市；
*   股本规模：总股本越小，协议转让的搜寻成本越高（需求侧正向），同时做市商满足持仓要求的成本越低（供给侧正向），越有可能选择做市；
*   股价：股价越高，做市降低每股交易成本的收益越高（需求侧正向），单笔交易的价差收入越高（供给侧正向），越有可能选择做市。

##### （2）做市商数量的影响因素（供给侧主导）

*   企业规模越大、交易越活跃，做市商的收入越高，愿意进入的做市商数量越多；
*   股价波动率越高，做市商风险成本越高，愿意进入的做市商数量越少；
*   总股本越小、股价越高，做市商的盈利空间越大、持仓成本越低，愿意进入的做市商数量越多。

##### （3）做市制度的效果假说

## 做市商能够通过连续报价、撮合交易降低交易的搜寻成本，因此企业从协议转让转为做市转让后，股票流动性会显著提升。

### 三、研究设计

#### 1. 样本与数据

样本区间为2014年6月做市制度落地至2015年底的新三板挂牌企业，剔除变量缺失样本，数据来自WIND数据库，使用STATA12进行计量分析。

#### 2. 变量定义

| 变量类型  | 变量名             | 变量含义                                                                                                                                                |
| ----- | --------------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| 被解释变量 | $Mmdum$         | 虚拟变量，企业选择做市转让取1，协议转让取0                                                                                                                              |
|       | $Mmnum$         | 企业的做市商数量，左截尾（最低为2家）                                                                                                                                 |
|       | $Liquidity$     | 股票流动性，三个衡量指标： 1. $Lnnonzero$：零收益天数占比的对数，值越高流动性越差 2. $Lnturnover$：换手率的对数，值越高流动性越好 3. $Lnpriceimpact$：Amihud非流动性指标的对数，即$|收益率|\times10^6/成交量$，值越高流动性越差 |
| 控制变量  | $Lnasset$       | 企业总资产对数，代表企业规模                                                                                                                                      |
|       | $Lnvolatility$  | 股票收益率波动率的对数                                                                                                                                         |
|       | $Lnoutstanding$ | 总股本的对数                                                                                                                                              |
|       | $Lnprice$       | 股票价格的对数                                                                                                                                             |
|       | $Balance$       | 股权制衡度，衡量股权分散程度                                                                                                                                      |


#### 3. 模型设定

##### （1）转让方式选择的检验模型（Logit模型）

$\displaystyle \begin{aligned} \text{Logit}(Mmdum_i =1 \mid Mmdum_i=0) &= a_0 +a_1 Liquidity_i +a_2 Lnasset_i +a_3 Lnvolatility_i \\ &+a_4 Lnoutstanding_i +a_5 Lnprice_i +a_6 Balance +\varepsilon_i \end{aligned}$

##### （2）做市商数量的检验模型（Tobit模型，解决左截尾问题）

$\displaystyle \begin{aligned} \text{Tobit}(Mmnum_i \leq 2) &= b_0 +b_1 Liquidity_i +b_2 Lnasset_i +b_3 Lnvolatility_i \\ &+b_4 Lnoutstanding_i +b_5 Lnprice_i +b_6 Balance +\varepsilon_i \end{aligned}$

##### （3）做市制度效果的检验模型（双重差分DID模型，解决自选择偏差）

## 选取2014年下半年为协议转让、2015年下半年转为做市转让的企业为处理组，始终采用协议转让的企业为对照组，构建模型： $\displaystyle \begin{aligned} Liquidity_{i} &= \alpha_0 +\alpha_1 Timedum_i +\alpha_2 Mmdum_i +\alpha_3 Timedum_i \times Mmdum_i \\ &+\alpha_4 Lnvolatility_i +\alpha_5 Lnoutstanding_i +\alpha_6 Lnprice_i +\varepsilon_i \end{aligned}$ 其中$Timedum$为时间虚拟变量，转为做市前取0，转为做市后取1；核心解释变量为交乘项$Timedum_i \times Mmdum_i$，其系数代表做市转让对流动性的净影响。

### 四、实证结果

#### 1. 转让方式与做市商数量的影响因素结果

*   流动性指标回归显示：流动性越好的企业越倾向于选择做市转让，说明供给侧的正向影响超过了需求侧的负向影响，做市商更愿意为流动性好的企业提供服务，最终这类企业更容易实现做市转让；
*   控制变量回归显示：总资产规模越大、总股本越小、股价越高的企业越倾向于选择做市转让，符合假说预期；
*   做市商数量回归显示：总资产规模越大、股价波动率越低的企业，做市商数量越多，符合供给侧假说，做市商更愿意为风险低、盈利空间大的企业提供服务。

#### 2. 做市制度的流动性效果结果

## 双重差分模型的交乘项系数显著：企业转为做市转让后，Amihud非流动性指标显著下降，换手率显著提升，说明做市转让确实能够显著改善股票流动性。该结果在通过倾向得分匹配（PSM）重新配对样本、更换流动性衡量指标、调整样本区间后依然稳健，排除了“流动性好的企业主动选择做市”的自选择干扰。

### 五、结论与政策建议

#### 核心结论

1.  新三板的做市服务供需共同决定了转让方式选择和做市商数量，不能仅从单一侧面解释；
2.  做市转让制度确实达到了政策预期，有效提升了新三板股票的流动性。

#### 政策建议

1.  降低做市商的做市成本，比如下调做市商最低持仓要求，鼓励银行、保险等更多机构参与做市，扩大做市服务供给；
2.  出台针对性政策降低中小市值企业的做市成本，提升企业选择做市转让的意愿；
3.  完善做市商的激励约束机制，加强对做市商报价行为的监管，提升做市服务质量，进一步改善新三板市场的整体流动性。
