## 📊 文献表格 - 科创板基础性制度改革能否提升市场定价效率?

| 维度 | 内容 |
| --- | --- |
| 论文标题 | 科创板基础性制度改革能否提升市场定价效率? |
| 作者 | 张宗新、吴钊颖 |
| 发表年份 | 2021 |
| 研究问题 | 探究科创板基础性制度改革能否提升市场定价效率，分析询价机制改革与价格限制改革对市场化定价的影响与作用路径。 |
| 研究方法 | 采用倾向得分匹配法构建对照组，构建实证回归模型，通过中介效应检验分析作用路径，开展稳健性检验验证结论。 |
| 主要发现 | 科创板基础性制度改革显著提升了市场定价效率，较好达成改革预期；询价机制改革实现了发行定价市场化，价格限制改革实现了交易定价市场化，二者共同优化了市场价格发现功能。 |
| 创新点 | 从市场定价效率视角搭建了科创板定价改革效应的论证框架，实证检验了两类核心改革的作用路径，为该领域研究提供了新的经验证据。 |
| 局限性 | 未提及 |
| 与本研究的关联 | 本文聚焦科创板注册制改革对定价效率的影响，可为同类研究提供研究框架与实证方法参考。 |
