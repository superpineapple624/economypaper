## 📊 文献表格 - 新三板挂牌企业退出做市：影响因素与经济后果

| 维度      | 内容                                                                                                                                              |
| ------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| 论文标题    | 新三板挂牌企业退出做市：影响因素与经济后果                                                                                                                           |
| 作者      | 胡妍、陈辉、莫志锴                                                                                                                                       |
| 发表年份    | 2019年                                                                                                                                           |
| 研究问题    | 从我国现行股票发行制度出发，实证检验新三板挂牌企业退出做市转让的影响因素，以及企业退出做市转让产生的经济后果。                                                                                         |
| 研究方法    | 采用LOGIT模型、混合回归模型、TOBIT模型开展基准回归，使用倾向性得分匹配法、分层分组回归进行稳健性检验，同时采用事件研究法计算平均累积超常收益率分析市场反应。                                                             |
| 主要发现    | 拟IPO企业的退出做市倾向显著更强，股票流动性不是挂牌企业退出做市的主要影响因素；退出做市对拟IPO企业的股票流动性存在显著负向影响，无IPO计划的企业退出做市后股票流动性显著上升；两类企业在退出做市日前后都存在显著为正的平均累积超常收益率。                       |
| 创新点     | 首次讨论新三板挂牌企业退出做市转让的影响因素，拓展了股票转让方式选择的相关研究；从股票流动性和证券价值角度检验退出做市的经济后果，为新三板市场交易机制与市场质量关系的研究提供了新的经验证据；从拟IPO企业面临的制度安排出发，丰富了企业IPO经济后果的研究，为新三板制度改革提供决策依据。 |
| 局限性     | 未提及                                                                                                                                             |
| 与本研究的关联 | 未提及                                                                                                                                             |
