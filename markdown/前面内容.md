# 一、引言

## （一）研究背景与问题提出

资本市场服务科技创新，既需要融资制度支持，也离不开有效的价格形成机制。科创板设立以来，围绕发行、交易和信息披露等环节持续推进改革。其中一个重要目标，就是提高市场资源配置效率，更好发挥资本市场对“硬科技”企业的价值发现功能。

科技企业的估值往往更难。其原因在于，这类企业通常研发投入高，不确定性强，信息解释难度也更大。在这种情况下，股票价格能否及时、充分地反映公司信息，就显得尤为重要。这个问题不仅关系到二级市场的交易质量，也会影响一级市场融资定价、企业再融资安排以及创新资源的持续配置。

做市商制度正是在这一背景下被引入科创板的重要交易机制改革。2022年10月31日，科创板正式启动股票做市交易业务，部分股票开始纳入做市标的名单。与以连续竞价为主的单一交易安排相比，做市商通过持续双边报价提供流动性，并在一定程度上缓冲短期交易冲击。按照理论预期，这一制度安排有助于降低交易摩擦，改善买卖价差，并提升价格发现效率。

对科技企业来说，这个问题更值得讨论。科技企业普遍存在较强的信息不对称，投资者之间的估值分歧也更明显。因此，做市商制度能否推动公司特质信息更快进入价格，并进一步改善定价效率，是一个具有现实针对性的问题。

不过，现有研究对于问题留有讨论空间。已有文献更多证明了做市商制度能够改善流动性、降低交易成本，或者表明科创板整体制度改革有助于提高市场质量。但直接围绕“做市商制度是否提升了科技企业定价效率”的证据，仍然相对有限。即使是已有的直接研究，其结论也常常表现为分指标、分阶段和分条件出现，而不是所有定价效率指标都同时表现出强而稳定的改善。

这说明，做市商制度的作用机制和作用边界仍有必要继续识别。基于此，本文聚焦一个更具体的问题：科创板做市商制度是否提高了科技企业的定价效率；如果提高，这种作用主要体现在哪些维度；又更可能出现在什么类型的企业中。围绕这一问题展开分析，有助于更准确地评价科创板交易制度改革的实际效果，也有助于理解在高不确定性、高信息摩擦的科技企业场景下，交易机制优化能否真正转化为价格发现效率的提升。

## （二）研究意义

本文的研究意义主要体现在理论和现实两个层面。

从理论层面看，关于做市商制度的研究已经积累了较多成果，但长期以来，讨论重点更多集中在流动性、买卖价差和市场波动上，对定价效率的直接讨论仍然不够充分。尤其是在科创板这样一个以科技企业为主体的市场中，企业的信息环境和估值逻辑都有较强特殊性。做市商制度究竟只是通过降低交易摩擦改善价格形成，还是能够进一步促进公司信息更充分地进入价格，仍然值得单独检验。

基于这一点，本文将定价效率作为核心结果变量，在流动性研究之外，进一步考察价格延迟、股价同步性和残差波动率等指标。这样的处理能够从更贴近价格形成过程的角度，补充做市商制度影响资本市场质量的研究视角。

从现实层面看，科创板承担着服务科技创新和推进资本市场改革试验的重要功能。做市商制度能否有效发挥作用，不只是交易机制优化的问题，也关系到科技企业估值形成、投资者定价行为以及资本市场支持创新的制度基础。如果实证结果表明做市商制度能够改善科技企业定价效率，那么这将为完善科创板做市商制度、优化做市标的遴选和配套激励约束机制提供经验支持。

即使制度效果表现出明显的分维度或分条件特征，这样的发现同样具有政策意义。因为它能够帮助监管部门和市场参与者更准确地理解制度适用范围，避免对做市商制度形成过高或过于单一的预期。

## （三）研究思路与研究方法

本文围绕“做市商制度是否改善科技企业定价效率”这一主线展开。研究对象为科创板上市公司，样本区间为2020年1月至2025年12月。结合科创板做市交易业务分批落地的制度安排，本文根据公司是否被纳入做市标的名单以及首次纳入做市日期，构造公司层面的制度冲击变量，并在公司月度面板数据上识别做市商制度实施前后的变化。

在研究设计上，本文首先采用多期双重差分模型，考察做市商制度对企业定价效率的平均影响。同时，控制公司固定效应和年月固定效应，以尽量剔除不随时间变化的个体差异和同期共同冲击。

在定价效率的测度上，本文以价格延迟为主指标，同时使用股价同步性和残差波动率作为替代指标。这样做的目的，是避免研究结论过度依赖单一口径。考虑到做市商制度的作用可能不会立刻显现，本文进一步构建事件研究模型，用于检验平行趋势假设，并观察制度影响的动态路径。

在此基础上，本文继续从三个方向推进实证分析。第一，围绕结果稳健性，采用替换被解释变量、对主要连续变量进行1%和99%分位数缩尾处理以及随机化安慰剂检验等方法，考察基准结论是否稳健。第二，围绕制度作用机制，结合现有文献对流动性、信息环境和波动特征的讨论，为结果解释提供机制层面的支撑。第三，围绕制度边界条件，分别按企业市值和信息透明度进行异质性分析，考察做市商制度是否在不同类型科技企业中表现出不同影响。

## （四）研究贡献与创新点

与已有研究相比，本文主要有以下几个方面的推进。

第一，本文将研究场景明确聚焦于科创板做市商制度这一具体制度安排，而不是笼统讨论注册制或市场化改革对市场质量的综合影响。科创板是我国服务科技创新的重要板块，科技企业在信息不对称、估值分歧和交易特征上都具有较强特殊性。因此，围绕科创板做市商制度展开讨论，可以使研究问题更集中，制度含义也更清楚。

第二，本文将考察重点从“做市商制度是否改善流动性”进一步推进到“做市商制度是否提升定价效率”。已有研究对流动性改善的证据相对较多，但流动性改善并不必然意味着价格发现效率同步提高。本文以价格延迟为主指标，以股价同步性和残差波动率为补充指标，更直接地考察股票价格中公司信息的反映程度，从而为评估做市商制度效果提供更贴近价格形成过程的证据。

第三，本文在识别策略上结合多期DID、事件研究、稳健性检验和异质性分析，尽量从多个维度提高结论的可信度。考虑到当前关于科创板做市商制度的直接证据仍然不多，而且制度效果本身可能呈现出分指标、分阶段和分条件的特点，本文不将研究目标设定为证明一种“全面而强烈”的统一结论，而是更关注制度效应是否存在、主要体现在哪些指标上，以及在什么类型的企业中更容易出现。这样的处理更符合科创板市场实际，也更有助于形成稳妥的经验判断。
